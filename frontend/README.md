# 動作環境について

このアプリケーションは、**モダンなWebブラウザ（フロントエンド）のみ**で動作する環境を想定して作られています。

主な特徴と動作環境の詳細は以下の通りです：

## 1. サーバーレス（静的ホスティングで動作）
* バックエンドサーバー（Node.js, Python, PHPなど）やデータベース（MySQLなど）は必要ありません。
* HTML, CSS, JS(TS) ファイルを配信できる環境（GitHub Pages, Vercel, Netlify, Amazon S3、またはローカルの簡易Webサーバー）があれば動作します。

## 2. データの保存先（ローカルストレージ）
* 作成した自己紹介カードのデータは、ブラウザの `localStorage` に保存されます。
* そのため、**同じブラウザ・同じ端末**でアクセスした時のみデータが保持されます。
* ※他のユーザー（他の新入生や上級生）とリアルタイムでデータを共有するには、Firebase (Firestore) や Supabase、GAS(Google Apps Script) などのバックエンドデータベースを連携させる拡張が必要です。現状は「自分だけの画面で動くプロトタイプ」となっています。

## 3. ビルド不要のモジュール読み込み（サンドボックス向け）
* `importmap` と CDN (esm.sh) を使用して、React などのライブラリをブラウザ上で直接読み込んでいます。
* `.tsx` (TypeScript + JSX) ファイルを使用しているため、このコード群がそのまま動くのは、**ブラウザ上でTypeScriptをオンザフライで変換（トランスパイル）できるプレビュー環境やサンドボックス環境**（現在のこのAIチャット画面のプレビュー機能など）です。
* 一般的なWebサーバーで公開する場合は、Vite や Webpack などのバンドラーを使って事前にビルド（通常のJavaScriptへの変換）を行う必要があります。

## まとめ
現状のコードは、「**現在のプレビュー環境（サンドボックス）で即座に動くように最適化された、ブラウザ完結型のモックアップアプリ**」です。
実際に新入生全員でアクセスして交流するためには、GASなどを利用してデータを保存・取得する処理を追加する必要があります。
